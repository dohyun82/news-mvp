# NewsBot 사용자 메뉴얼

NewsBot은 네이버 뉴스를 자동으로 수집하고, AI로 요약한 뒤 카테고리별로 정리하여 그룹웨어 게시판에 복사·붙여넣기할 수 있도록 돕는 도구입니다. 이 메뉴얼은 NewsBot을 처음 사용하는 사용자를 위한 단계별 가이드입니다.

## 시작하기

### 1. 서버 접속

브라우저에서 다음 주소로 접속합니다:
```
http://127.0.0.1:5001/review
```

### 2. 초기 화면 구성

검토 페이지(`/review`)는 두 개의 주요 영역으로 구성되어 있습니다:

- **왼쪽 영역**: 전체 뉴스 목록
  - 수집된 모든 뉴스가 표시됩니다
  - 뉴스 수집 버튼이 있습니다

- **오른쪽 영역**: 선택된 뉴스 목록
  - 선택한 뉴스가 카테고리별로 표시됩니다
  - 카테고리: 그룹사 / 유통·F&B·급식 / 플랫폼·테크 / 경쟁사·복지·HR 트렌드 / 시장·소비 트렌드

## 주요 기능

### 1. 뉴스 수집
- 네이버 뉴스 API를 통해 설정된 키워드로 뉴스를 검색합니다
- 수집된 뉴스는 모두 "미분류" 상태이며, 검토 화면에서 직접 카테고리를 지정합니다
- 중복·광고성 기사는 1차 자동 필터링됩니다

### 2. 뉴스 요약
- OpenAI AI를 사용하여 뉴스를 3~5줄로 요약합니다
- 개별 요약 또는 일괄 요약이 가능합니다

### 3. 뉴스 선택 및 분류
- 드래그 앤 드롭으로 뉴스를 5개 카테고리에 직접 분류합니다(수동)
- 불필요한 뉴스는 삭제할 수 있습니다

### 4. 복사하기 (그룹웨어 게시판)
- 선택·분류된 뉴스를 카테고리별로 정리한 내용으로 클립보드에 복사합니다
- 그룹웨어 게시판에 붙여넣어 게시합니다

## 검토 페이지 사용법

### 1. 뉴스 수집하기

**단계:**
1. 왼쪽 상단의 **"뉴스 수집"** 버튼을 클릭합니다
2. 수집이 완료될 때까지 기다립니다 (로딩 화면 표시)
3. 수집된 뉴스가 왼쪽 "전체 뉴스 목록"에 표시됩니다

**주의사항:**
- 뉴스가 이미 수집되어 있으면 "뉴스 수집" 버튼이 비활성화됩니다
- 새로운 뉴스를 수집하려면 먼저 **"초기화"** 버튼을 클릭하세요

### 2. 뉴스 검토 및 선택하기

**드래그 앤 드롭으로 선택:**
1. 왼쪽 목록에서 발송할 뉴스를 찾습니다
2. 뉴스 카드를 마우스로 드래그합니다
3. 오른쪽의 원하는 카테고리 영역에 드롭합니다
   - 예: "그룹사 뉴스" 영역에 드롭하면 그룹사 카테고리로 분류됩니다

**뉴스 삭제하기:**
- 불필요한 뉴스는 각 카드의 **"삭제"** 버튼을 클릭합니다

**선택 해제하기:**
- 선택된 뉴스를 다시 왼쪽으로 옮기려면 **"선택 해제"** 버튼을 클릭합니다

**카테고리 변경하기:**
- 이미 선택된 뉴스의 카테고리를 변경하려면 다른 카테고리 영역으로 드래그 앤 드롭합니다

### 3. 뉴스 요약하기

**개별 요약:**
1. 선택된 뉴스 카드에서 **"뉴스 요약"** 버튼을 클릭합니다
2. 요약이 완료되면 카드 하단에 요약 내용이 표시됩니다

**일괄 요약:**
1. 오른쪽 상단의 **"뉴스 요약"** 버튼을 클릭합니다
2. 요약되지 않은 모든 뉴스가 순차적으로 요약됩니다
3. 진행 상황이 로딩 화면에 표시됩니다

**주의사항:**
- 요약이 완료되지 않은 뉴스가 있으면 "뉴스 요약" 버튼이 활성화됩니다
- 선택된 뉴스가 있으면 "복사하기" 버튼이 활성화됩니다

### 4. 복사하기 (그룹웨어 게시판)

**단계:**
1. 선택·분류·요약이 끝났는지 확인합니다
2. 오른쪽 상단의 **"복사하기"** 버튼을 클릭합니다
3. 카테고리별로 정리된 내용이 클립보드에 복사되고, 하단 미리보기에 표시됩니다
4. 그룹웨어 게시판 작성 화면에 붙여넣기(Ctrl/Cmd+V)합니다

**주의사항:**
- 브라우저 보안상 자동 복사가 막히면(HTTP 접속 등), 하단 미리보기에서 직접 복사(Ctrl/Cmd+C)하세요
- 붙여넣을 게시판 에디터에 따라 서식이 달라질 수 있습니다

### 5. 초기화하기

**단계:**
1. 왼쪽 상단의 **"초기화"** 버튼을 클릭합니다
2. 모든 수집된 뉴스가 삭제됩니다
3. 새로운 뉴스를 수집할 수 있게 됩니다

**주의사항:**
- 초기화하면 선택된 뉴스와 요약 내용도 모두 삭제됩니다
- 되돌릴 수 없으므로 신중하게 사용하세요

## 설정 페이지 사용법

설정 페이지(`/settings`)에서는 뉴스 수집에 사용되는 키워드와 옵션을 관리할 수 있습니다.

### 설정 페이지 접속

- 검토 페이지 우측 상단의 **"설정"** 버튼을 클릭합니다
- 또는 브라우저에서 `http://127.0.0.1:5001/settings`로 직접 접속합니다

### 1. 뉴스 검색 키워드 관리

**목적:** 네이버 뉴스 검색에 사용할 키워드를 설정합니다.

**사용법:**
1. "키워드 추가" 입력란에 키워드를 입력합니다
   - 예: `현대백화점, 식권대장, vendys`
   - 쉼표로 구분하여 여러 키워드를 한 번에 추가할 수 있습니다
2. **"추가"** 버튼을 클릭합니다
3. 등록된 키워드는 하단에 태그로 표시됩니다
4. 키워드를 삭제하려면 태그의 **×** 버튼을 클릭합니다

**예시 키워드:**
- 그룹사 관련: `현대백화점`, `식권대장`, `vendys`, `현대이지웰`
- 업계 관련: `모바일 식권`, `푸드테크`, `식대 정산`

### 2. 최대 수집 뉴스 개수 설정

**목적:** 한 번에 수집할 최대 뉴스 기사 개수를 제한합니다.

**사용법:**
1. "최대 수집 개수" 입력란에 숫자를 입력합니다
2. 기본값: `30`
3. 1 이상의 숫자만 입력 가능합니다

**권장값:**
- 일일 작업량에 따라 조정하세요
- 너무 많으면 검토 시간이 길어집니다
- 너무 적으면 중요한 뉴스를 놓칠 수 있습니다

### 3. 뉴스 조회 시간 설정

**목적:** 설정한 시간 이전에 발행된 뉴스는 수집하지 않도록 제한합니다.

**사용법:**
1. "최대 조회 시간 (시간)" 입력란에 숫자를 입력합니다
2. 기본값: `24` (24시간 이내 뉴스만 수집)
3. `0`으로 설정하면 시간 제한이 없습니다

**예시:**
- `24`: 최근 24시간 이내 뉴스만 수집
- `48`: 최근 48시간 이내 뉴스만 수집
- `0`: 시간 제한 없음 (모든 뉴스 수집)

### 4. 설정 저장하기

**단계:**
1. 모든 설정을 변경한 후
2. 페이지 하단의 **"설정 저장"** 버튼을 클릭합니다
3. "설정이 저장되었습니다." 메시지가 표시되면 완료입니다

**주의사항:**
- 설정을 저장하지 않으면 변경 사항이 적용되지 않습니다
- 저장된 설정은 다음 뉴스 수집부터 적용됩니다
- 설정은 브라우저의 로컬 스토리지와 서버에 저장됩니다

## 일일 작업 흐름

뉴스 담당자가 매일 수행하는 권장 작업 순서입니다.

### 오전 9시 ~ 10시 (목표 시간)

**1단계: 뉴스 수집 (약 1분)**
- 검토 페이지에서 **"뉴스 수집"** 버튼 클릭
- 수집 완료까지 대기

**2단계: 뉴스 검토 및 선택 (약 10분)**
- 왼쪽 목록에서 발송할 뉴스 확인
- 불필요한 뉴스는 **"삭제"** 버튼으로 제거
- 발송할 뉴스를 드래그 앤 드롭으로 카테고리별로 선택
- 카테고리가 잘못 분류된 경우 다른 카테고리로 이동

**3단계: 뉴스 요약 (약 5분)**
- 오른쪽 상단의 **"뉴스 요약"** 버튼 클릭
- 모든 뉴스 요약 완료까지 대기
- 개별 요약이 필요한 경우 각 카드의 **"뉴스 요약"** 버튼 사용

**4단계: 최종 검토 (약 5분)**
- 선택된 뉴스와 요약 내용을 최종 확인
- 필요시 뉴스 추가/삭제 또는 요약 재생성

**5단계: 복사 후 게시 (약 1분)**
- **"복사하기"** 버튼 클릭
- 그룹웨어 게시판에 붙여넣은 뒤 게시

**총 소요 시간: 약 20~30분**

## 자주 묻는 질문 (FAQ)

### Q1. 뉴스 수집 버튼이 비활성화되어 있습니다.

**A:** 이미 뉴스가 수집되어 있는 상태입니다. 새로운 뉴스를 수집하려면:
1. **"초기화"** 버튼을 클릭하여 기존 뉴스를 삭제하거나
2. 기존 뉴스를 모두 검토하고 발송한 후 초기화하세요

### Q2. 뉴스 요약 버튼이 비활성화되어 있습니다.

**A:** 다음 중 하나의 경우입니다:
- 선택된 뉴스가 없습니다 → 뉴스를 먼저 선택하세요
- 모든 뉴스가 이미 요약 완료되었습니다 → "복사하기" 버튼을 사용하세요

### Q3. 복사하기 버튼이 비활성화되어 있습니다.

**A:** 선택된 뉴스가 없으면 비활성화됩니다. 왼쪽 목록의 뉴스를 카테고리 영역으로 드래그해 선택하세요.

### Q4. 뉴스가 잘못된 카테고리로 분류되었습니다.

**A:** 드래그 앤 드롭으로 올바른 카테고리로 이동할 수 있습니다:
1. 선택된 뉴스 카드를 드래그합니다
2. 올바른 카테고리 영역에 드롭합니다
3. 카테고리가 자동으로 변경됩니다

### Q5. 설정을 변경했는데 적용이 안 됩니다.

**A:** 다음을 확인하세요:
1. **"설정 저장"** 버튼을 클릭했는지 확인
2. 저장 후 새로운 뉴스를 수집해야 설정이 적용됩니다
3. 브라우저를 새로고침해보세요

### Q6. 뉴스 요약이 생성되지 않습니다.

**A:** 다음을 확인하세요:
1. OpenAI API 키가 올바르게 설정되어 있는지 확인
2. 네트워크 연결 상태 확인
3. 브라우저 콘솔에서 에러 메시지 확인
4. 잠시 후 다시 시도해보세요

### Q7. 복사가 되지 않습니다.

**A:** 브라우저 보안상 클립보드 접근이 막힌 경우입니다(HTTP 접속 등). 하단 미리보기 영역에서 직접 전체 선택 후 복사(Ctrl/Cmd+C)하세요.

### Q8. 뉴스가 너무 많이 수집됩니다.

**A:** 설정 페이지에서 다음을 조정하세요:
1. **"최대 수집 뉴스 개수"**를 줄입니다 (예: 30 → 20)
2. **"최대 조회 시간"**을 줄입니다 (예: 24시간 → 12시간)
3. 불필요한 키워드를 제거합니다

### Q9. 원하는 뉴스가 수집되지 않습니다.

**A:** 설정 페이지에서 다음을 확인하세요:
1. 검색 키워드에 원하는 키워드가 포함되어 있는지 확인
2. 키워드를 추가한 후 **"설정 저장"**을 클릭했는지 확인
3. 새로운 뉴스를 수집해보세요

### Q10. 초기화하면 모든 데이터가 삭제되나요?

**A:** 네, 초기화하면 다음이 모두 삭제됩니다:
- 수집된 모든 뉴스
- 선택된 뉴스
- 요약 내용

**주의:** 설정(키워드 등)은 삭제되지 않습니다.

## 문제 해결

### 문제 1: 페이지가 로드되지 않습니다

**해결 방법:**
1. 서버가 실행 중인지 확인하세요
2. 브라우저 주소창에 올바른 URL을 입력했는지 확인하세요 (`http://127.0.0.1:5001/review`)
3. 브라우저를 새로고침해보세요 (Ctrl+F5 또는 Cmd+Shift+R)
4. 다른 브라우저에서 시도해보세요

### 문제 2: 뉴스 수집이 실패합니다

**해결 방법:**
1. 네이버 API 키가 올바르게 설정되어 있는지 확인하세요
2. 네트워크 연결 상태를 확인하세요
3. 브라우저 콘솔(F12)에서 에러 메시지를 확인하세요
4. 서버 로그를 확인하세요

### 문제 3: 뉴스 요약이 느립니다

**원인:**
- OpenAI API 호출에 시간이 걸립니다
- 뉴스가 많을수록 시간이 더 걸립니다

**해결 방법:**
1. 요약 중에는 페이지를 닫지 마세요
2. 네트워크 연결 상태를 확인하세요
3. 뉴스 개수를 줄여보세요 (설정에서 "최대 수집 뉴스 개수" 조정)

### 문제 4: 드래그 앤 드롭이 작동하지 않습니다

**해결 방법:**
1. 브라우저를 새로고침해보세요
2. 다른 브라우저에서 시도해보세요
3. 마우스 드래그가 정상적으로 작동하는지 확인하세요
4. 터치스크린 기기에서는 드래그 앤 드롭이 제한될 수 있습니다

### 문제 5: 설정이 저장되지 않습니다

**해결 방법:**

3. 브라우저를 새로고침한 후 다시 시도해보세요
4. 다른 브라우저에서 시도해보세요
